<div className="text-sm font-mono">
  <span className="font-semibold text-blue-500">GET</span> `/inbox`
</div>

Returns every stored inbound message for the signed-in user's company, newest first. These are the raw emails received through the inbound webhook — one `messages` row per delivery, stored verbatim before any classification runs.

## Authorization

Requires a platform JWT. See [Authentication](/authentication).

```
Authorization: Bearer <supabase-access-token>
```

This endpoint takes no query parameters and no request body.

## Response

Returns an array of `messages` rows, newest first.

<ResponseField name="[].id" type="string">
  The message's unique id.
</ResponseField>

<ResponseField name="[].company_id" type="string">
  The company that received the message.
</ResponseField>

<ResponseField name="[].company_email_id" type="string | null">
  The forwarding address the mail hit (→ `company_emails`). Null if it could not be matched.
</ResponseField>

<ResponseField name="[].channel" type="string">
  The delivery channel, e.g. `email`.
</ResponseField>

<ResponseField name="[].provider_message_id" type="string">
  The provider's message id. Unique per channel — powers idempotency.
</ResponseField>

<ResponseField name="[].from_address" type="string">
  The sender's email address.
</ResponseField>

<ResponseField name="[].to_address" type="string">
  The address the message was delivered to (the `@in.quotenow-app.com` forwarding address).
</ResponseField>

<ResponseField name="[].subject" type="string | null">
  The email subject.
</ResponseField>

<ResponseField name="[].text" type="string | null">
  The message body — plain text, or an HTML fallback when no plain-text part is present.
</ResponseField>

<ResponseField name="[].media_urls" type="array">
  Attachment metadata objects.
</ResponseField>

<ResponseField name="[].raw_payload" type="object">
  The full provider JSON, stored verbatim.
</ResponseField>

<ResponseField name="[].triggered_quotation" type="boolean | null">
  The classifier's verdict on whether the message should produce a quotation. `null` means unclassified.
</ResponseField>

<ResponseField name="[].triggered_quotation_metadata" type="object | null">
  The classifier's reasoning: `{ confidence, reason, model }`.
</ResponseField>

<ResponseField name="[].sent_at" type="string | null">
  The email's own date header.
</ResponseField>

<ResponseField name="[].message_id" type="string | null">
  Threading pointer to another message (→ `messages`) — the message this one replies to.
</ResponseField>

<ResponseField name="[].created_at" type="string">
  ISO 8601 timestamp of when the message was stored.
</ResponseField>

## Errors

<ResponseField name="401" type="Invalid or expired token">
  The JWT is missing, malformed, or expired.
</ResponseField>

<ResponseField name="404" type="No company linked to this user">
  The authenticated user has no company.
</ResponseField>
