<div className="text-sm font-mono">
  <span className="font-semibold text-blue-500">GET</span> `/integrations`
</div>

Returns all inbound forwarding addresses for the signed-in user's company, newest first.

## Authorization

Requires a platform JWT. See [Authentication](/authentication).

```
Authorization: Bearer <supabase-access-token>
```

## Response

Returns an **array** of `company_emails` rows, ordered by creation time with the newest first.

<ResponseField name="[].id" type="string">
  The address's unique id.
</ResponseField>

<ResponseField name="[].company_id" type="string">
  The company that owns the address.
</ResponseField>

<ResponseField name="[].inbound_email" type="string">
  The generated `@in.quotenow-app.com` forwarding address.
</ResponseField>

<ResponseField name="[].created_at" type="string">
  ISO 8601 creation timestamp.
</ResponseField>

## Errors

<ResponseField name="401" type="Invalid or expired token">
  The JWT is missing, malformed, or expired.
</ResponseField>

<ResponseField name="404" type="No company linked to this user">
  The authenticated user has no company.
</ResponseField>

<RequestExample>
```bash cURL
curl https://oblea-backend-prod-ye2cpok36a-uc.a.run.app/integrations \
  -H "Authorization: Bearer <supabase-access-token>"
```

```javascript JavaScript
const res = await fetch(`${API_URL}/integrations`, {
  headers: {
    Authorization: `Bearer ${token}`,
  },
});
const addresses = await res.json();
```
</RequestExample>
