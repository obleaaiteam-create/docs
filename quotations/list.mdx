<div className="text-sm font-mono">
  <span className="font-semibold text-blue-500">GET</span> `/quotations`
</div>

Returns all quotations for the signed-in user's company, newest first.

## Authorization

Requires a platform JWT. See [Authentication](/authentication).

```
Authorization: Bearer <supabase-access-token>
```

This endpoint takes no query parameters and no request body.

## Response

Returns an array of `quotation` rows, newest first.

</ResponseField>

<ResponseField name="[].company_id" type="string">

## Errors

<ResponseField name="401" type="Invalid or expired token">
  The JWT is missing, malformed, or expired.
</ResponseField>

</ResponseField>
